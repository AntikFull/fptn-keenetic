# FPTN для роутеров Keenetic (Entware)

Специализированный форк VPN-технологии FPTN, ориентированный на развертывание и использование клиента на домашних маршрутизаторах Keenetic. 

FPTN — это VPN-технология, созданная с нуля для безопасного, быстрого и устойчивого к блокировкам (DPI) соединения, позволяющего обходить сетевую фильтрацию. Данный репозиторий содержит интерактивный установщик и веб-панель управления клиентом для роутера.

---

## Быстрая установка в один клик

### 🚀 Стандартная установка:
Скачайте скрипт установщика во временный файл и запустите его (это необходимо для корректной работы интерактивного ввода в терминале):
```bash
curl -fsSL -o /tmp/install.sh https://raw.githubusercontent.com/AntikFull/fptn-keenetic/master/deploy/keenetic/install.sh && sh /tmp/install.sh
```

### ⚡ Зеркало для РФ (в случае блокировок GitHub):
Если ваш провайдер или ТСПУ блокирует GitHub (зависает на raw.githubusercontent.com), используйте быстрое CDN-зеркало:
```bash
curl -fsSL -o /tmp/install.sh https://cdn.jsdelivr.net/gh/AntikFull/fptn-keenetic@master/deploy/keenetic/install.sh && sh /tmp/install.sh
```
*(Или через `wget`, если на роутере нет curl:)*
```bash
wget -O /tmp/install.sh https://cdn.jsdelivr.net/gh/AntikFull/fptn-keenetic@master/deploy/keenetic/install.sh && sh /tmp/install.sh
```

### Что делает установщик:
1. **Устанавливает пакеты:** веб-сервер `lighttpd`, `php8-cgi` (для веб-интерфейса), `curl` и сертификаты.
2. **Определяет архитектуру процессора:** автоматически определяет вашу аппаратную платформу (`aarch64`, `armv7` или `mipsel`) и скачивает подходящий статический бинарник клиента FPTN.
3. **Запрашивает параметры:** имя интерфейса, порт для веб-панели (по умолчанию `8088`) и ваш токен подписки.
4. **Создает и настраивает TUN-интерфейс в KeeneticOS:** автоматически регистрирует интерфейс типа `OpkgTun` с флагами `ip global` и подстройкой TCP MSS, благодаря чему роутер видит туннель как полноценное интернет-подключение.
5. **Настраивает автозапуск:** прописывает службу в `/opt/etc/init.d/S53fptn-client`.

## Использование и веб-панель управления

После завершения установки откройте веб-панель в вашем браузере:
```
http://192.168.1.1:8088/fptn/
```
*(Если вы выбрали другой порт при установке, укажите его вместо `8088`)*.

В веб-панели вы можете:
* Сохранить/обновить токен подписки FPTN.
* Посмотреть список доступных премиум-серверов.
* Выбрать предпочтительный сервер или оставить автовыбор.
* Запускать, останавливать и перезапускать службу VPN-клиента.
* Отслеживать статус службы и TUN-интерфейса.

## Настройка маршрутизации в Keenetic

Вы можете заворачивать трафик домашних устройств в туннель FPTN штатными средствами KeeneticOS:

1. **Маршрутизация устройств (Политики):**
   Перейдите в веб-интерфейсе роутера в раздел **«Сетевые правила» -> «Приоритеты подключений»**. В списке подключений появится новое соединение **Fptn**. Перетащите его в нужную политику маршрутизации (например, создайте отдельную политику для обхода блокировок и добавьте туда нужные домашние устройства).

2. **DNS-маршрутизация доменов:**
   В разделе **«Сетевые правила» -> «Маршрутизация»** (или «Настройка DNS-маршрутов») вы можете привязывать определенные группы доменов напрямую к интерфейсу **Fptn** (например, `OpkgTun1`).

Подробные инструкции по ручной настройке и маршрутизации доступны в файле [deploy/keenetic/README.md](deploy/keenetic/README.md).
